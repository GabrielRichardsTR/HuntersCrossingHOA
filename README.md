# Hunters Crossing HOA

Website files for the Hunters Crossing Homeowners Association — Hunters Trail, Ann Arbor, MI.

## What's in this repository

| File | What it is | Who sees it |
|---|---|---|
| `index.html` | The public neighborhood website — welcome info, community FAQs, contact | Anyone with the link |
| `board-portal.html` | The board's private management tool — resident directory, violations, dues, documents | Board members only (not yet password-protected — see note below) |

## How to view the live site

Once GitHub Pages is turned on (Settings → Pages), both pages are available at:

- Public site: `https://gabrielrichardstr.github.io/HuntersCrossingHOA/`
- Board portal: `https://gabrielrichardstr.github.io/HuntersCrossingHOA/board-portal.html`

## How to make changes

1. Click on the file you want to edit (`index.html` or `board-portal.html`)
2. Click the pencil icon (top right) to edit
3. Make your changes
4. Scroll down, add a short description of what you changed, click "Commit changes"
5. The live site updates automatically within about a minute

No software or downloads needed — this can all be done from a web browser.
